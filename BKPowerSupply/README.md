# BKPowerSupply
BK powersupply control

The control lib uses the pyserial module: https://github.com/pyserial/pyserial?tab=readme-ov-file

The GUI uses the python tkinter: https://docs.python.org/3/library/tkinter.html

BK Model 1697B (1-channel DC PS) documentation: https://www.bkprecision.com/products/power-supplies/1697B

BK Model BK9140 (3-channel DC PS) documentation: https://www.bkprecision.com/products/power-supplies/9140
